StateGuy

a minimalist JavaScript state management library

Hello

This is the homepage for StateGuy, a boring and un-innovative global state management library for simple webpages.

I don't know how useful it might be for serious web development, but it's a good place to start if you're interested in recreating parts of your favorite framework as a learning exercise, or if you're making a small personal site that uses limited bandwidth.
